# LLM4VLM 数据集说明

## 数据集概述

本数据集包含使用大语言模型（LLM）生成的中文 VLN 导航指令，以及相关的视觉特征和标注数据。

## 数据目录结构

```
data/
├── r2r_enhanced/           # R2R 增强数据集
│   ├── train.json          # 训练集（约 1,000 样本）
│   ├── val.json            # 验证集（约 200 样本）
│   └── vocabulary.json     # 中文词表（97 字符）
├── r2r_raw/                # 原始 R2R 数据
├── sample_paths.json       # 路径样本示例
└── evaluation_r2r/         # 模型评估结果
    ├── model_evaluation_metrics.json      # 聚合指标
    └── model_evaluation_detailed.json     # 详细结果
```

## 数据格式

### 训练/验证数据格式

每个 JSON 样本包含以下字段：

```json
{
  "path_id": "r2r_val_00000",
  "instruction": "直走 3 米，然后左转，看到过道就到了。",
  "instruction_ids": [12, 34, 56, ...],
  "visual_features": [[...], [...], ...],
  "candidate_directions": [[...], [...], ...],
  "target_action": 11
}
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `path_id` | string | 路径唯一标识符 |
| `instruction` | string | 中文导航指令 |
| `instruction_ids` | list[int] | Tokenized 指令 IDs（已添加 [CLS] 和 [SEP]） |
| `visual_features` | list[list[float]] | ResNet-50 视觉特征 (num_views × 2048) |
| `candidate_directions` | list[list[float]] | 候选方向特征 (36 × 2048) |
| `target_action` | int | 目标动作 ID (0-35) |

### 动作空间

36 个动作分为 3 组：

| 动作 ID 范围 | 类型 | 说明 |
|-------------|------|------|
| 0-11 | 左转 | 逆时针旋转 (0°, 30°, ..., 330°) |
| 12-23 | 右转 | 顺时针旋转 (0°, 30°, ..., 330°) |
| 24-35 | 直走 | 向前移动 (0°, 30°, ..., 330°) |

### 词表说明

词表包含 97 个字符：

| 类别 | 字符数 | 示例 |
|------|--------|------|
| 特殊标记 | 4 | [PAD], [UNK], [CLS], [SEP] |
| 方向词 | 8 | 左，右，前，后，直，转，走 |
| 位置词 | 8 | 上，下，进，出，过，经，到，在 |
| 名词 | 20+ | 楼，梯，廊，道，门，口... |
| 数字 | 12 | 0-9, 十，百 |
| 标点 | 3 | ，，。,、 |

## 视觉特征

### 特征提取

- **骨干网络**: ResNet-50 (预训练于 ImageNet)
- **特征维度**: 2048
- **视角数量**: 可变 (通常 5-16 个视角)
- **投影**: 2048 → 256 (用于模型输入)

### 候选方向

- **数量**: 36 个候选方向
- **表示**: 每个方向 2048 维特征
- **覆盖**: 360 度全向覆盖 (每 10 度一个候选)

## 数据生成方法

### LLM 指令生成

使用 Qwen-3.5-Plus 模型生成中文指令：

1. **输入**: 路径描述（地标、方向、距离）
2. **Prompt**: 结构化提示模板
3. **输出**: 自然中文导航指令

### 质量评估

| 维度 | LLM 生成 | 机器翻译 |
|------|----------|----------|
| 优秀率 | 76.7% | 20.0% |
| 平均分 (1-5) | 4.61 | 4.28 |
