# 中文 VLN + 跨语言迁移研究计划

> 研究方向：多语言/跨文化 Vision-and-Language Navigation
> 目标：构建首个中文 VLN 数据集 + 跨语言迁移方法
> 预期产出：ACL/EMNLP/CVPR 论文

---

## 一、研究动机

### 1.1 问题背景

视觉语言导航（VLN）要求智能体根据自然语言指令在室内环境中导航。然而：

- **语言单一性**：现有研究几乎全部基于英语指令
- **跨语言泛化差**：英语训练的模型在其他语言上性能大幅下降
- **中文研究空白**：尚无大规模中文 VLN 数据集和方法

### 1.2 研究问题

1. 中文空间指令与英语有何系统性差异？
2. 如何利用英语数据辅助中文 VLN 学习？
3. 能否实现零样本/少样本跨语言 VLN 迁移？

### 1.3 研究意义

| 维度 | 意义 |
|------|------|
| 理论 | 揭示语言类型对空间导航的影响 |
| 方法 | 提出跨语言 VLN 迁移框架 |
| 资源 | 发布首个中文 VLN 数据集 |
| 应用 | 服务中文智能家居、服务机器人 |

### 3.2 任务 1：中文 VLN 数据集构建

#### 3.2.1 仿真环境选择

| 平台 | 优势 | 劣势 |
|------|------|------|
| **Habitat** | 高效、支持 MatterPort3D | 需配置 |
| AI2-THOR | 交互性强 | 计算开销大 |
| SAPIEN | 物理真实 | 学习曲线陡 |

**推荐**：Habitat + MatterPort3D

#### 3.2.2 数据规模设计

| 划分 | 场景数 | 路径数 | 指令数 |
|------|--------|--------|--------|
| 训练集 | 60 | 4,000 | 12,000 |
| 验证集 | 10 | 800 | 2,400 |
| 测试集 | 20 | 1,200 | 3,600 |
| **总计** | **90** | **6,000** | **18,000** |

#### 3.2.3 指令标注规范

**指令设计要求**：
- 长度：30-80 词
- 步数：4-8 个动作
- 包含：地标、方向、距离信息
- 自然：避免机械化表达

**示例**：
```
从桌子旁边出发，直走穿过拱门，
看到楼梯后右转，走到红色沙发前停下。
```

#### 3.2.4 标注平台

- 自研标注工具（基于 Habitat WebRTC）
- 或使用现有平台：Scale AI / 亚马逊 Mechanical Turk

### 3.3 任务 2：跨语言 VLN 方法

#### 3.3.1 基线模型 1：翻译基线

```
中文指令 → 机器翻译 → 英文 VLN 模型 → 动作
```

- 使用预训练 VLN 模型（HAMT/VLN-BERT）
- 测试翻译质量影响

#### 3.3.2 基线 2：多语言编码器

```
中文指令 → mBERT/XLM-R → 共享编码器 → 动作
```

- 在英文数据预训练
- 在中文数据微调

#### 3.3.3 我们的方法：跨语言适配器

```
┌─────────────────────────────────────────┐
│           English VLN Encoder           │
├─────────────────────────────────────────┤
│    Language Adapter (中文)   │
│    - 参数高效（仅 2-5M）                │
│    - 即插即用                          │
└─────────────────────────────────────────┘
```

**核心创新**：
- 轻量语言适配器
- 跨语言对比学习
- 语言不变表示学习

#### 3.3.4 零样本探索

```
英文训练 → 语言对齐 → 中文零样本推理
```

利用多语言 LLM 的空间知识迁移

### 3.4 任务 3：分析与评估

#### 3.4.1 评估指标

| 指标 | 说明 |
|------|------|
| **SR** | 成功率（到达目标） |
| **SPL** | 成功率 × 路径效率 |
| **SDTW** | 动态时间弯曲相似度 |
| **Oracle** | 上限估计 |

#### 3.4.2 语言差异分析

- 指令长度分布
- 空间词汇频率
- 句法结构对比

#### 3.4.3 错误分析

- 导航失败类型分类
- 语言误解 vs 视觉误解
- 跨语言迁移错误
